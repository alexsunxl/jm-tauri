#  jm-tauri 客户端 🧭

- 🧩 Tauri + React 客户端支持 Windows / macOS / Linux / Android(apk)
- 🛠️ 技术栈：TypeScript、React、Vite、Tailwind CSS、Tauri (Rust)
- ⚠️ 仅供技术研究，请勿用于其他用途
- 💬 如有问题欢迎提 ISSUE

## 功能概览 ✨
- 登录/搜索/详情/阅读
- 在线收藏 / 本地收藏 / 浏览记录
- 分类与排行
- 阅读进度记录、继续阅读
- 代理设置与 API 域名管理（含测速）
- 缓存管理与状态显示
- 下载与本地缓存（按平台写入可写目录）

## 阅读进度（亮点） 📌
- 自动记录到章节/页级别
- 列表内可直接继续阅读
- 支持本地收藏与历史记录的进度联动
